## Default Permission

Default permissions for the liquid-glass plugin

#### This default permission set includes the following:

- `allow-is-glass-supported`
- `allow-set-liquid-glass-effect`

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>

`liquid-glass:allow-is-glass-supported`

</td>

</td>
</tr>

<tr>
